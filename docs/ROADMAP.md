# Roadmap

## M0 Foundation

- Monorepo 和构建基线；
- 产品章程、架构、DSL、表单和连接器协议；
- Java 领域、应用、Engine SPI 和 Flowable 适配骨架；
- TypeScript contracts、process-dsl 和 form-schema 包；
- CI、依赖和安全基线。

## M1 First Vertical Slice

以采购付款审批为样板：

- 创建流程、动态表单和发布版本；
- 发起、部门负责人审批和财务会签；
- PC 待办与审批详情；
- 移动端待办与审批操作；
- 任意驳回、催办、抄送和业务回调；
- 全链路审计和幂等。

## M2 Designer and Forms

- 树状审批设计器；
- 条件/并行分支和静态检查；
- 表单设计器、联动、公式和节点权限；
- DSL 编译器和流程模拟器。

## M3 Collaboration

- 顺序/并行会签、或签、票签、权重；
- 前后加签、减签、转办、委派和协办；
- 代理、离职转办和流程交接；
- 评论、@、附件、已阅和多渠道消息。

## M4 Operations

- 工作日历、SLA、超时升级和自动处理；
- 运维控制台、Job 重试、流程跳转和补偿；
- 流程版本迁移和批次预演；
- 监控、告警、容量和数据清理。

## M5 Ecosystem and AI

- RuoYi 5/6、钉钉、飞书连接器；
- SDK、模板市场和第三方组件；
- AI 材料检查、摘要、风险和意见建议；
- 受控自动审批和 AI 审计。
